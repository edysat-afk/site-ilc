## Detalhamento por Página

### HOME (/)
- **Seções:**
  1. Hero/Banner principal com CTA "Saiba Mais"
  2. Serviços em destaque (4 cards)
  3. Seção de blog/notícias (3 artigos em destaque)
  4. CTA secundário para contato
- **Elementos:** Logo, menu, rodapé com copyright

### MASTERMIND (/mastermind-infraestrutura-e-logistica/)
- **Descrição:** Programa premium de eventos/sessões sobre infraestrutura
- **Edições:** Infra 2023, Infra 2024, Infra 2025, Infra 2026, Portos
- **Conteúdo protegido:** Acesso via login
- **Materiais:** Páginas de sessão + PDFs para download
- **Volume:** ~120+ páginas de sessões individuais
